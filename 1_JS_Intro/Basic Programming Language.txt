Build Blocks of Programming Language

-Variables

Variables => used to store some data in a container. e.g => container (Variable) = 3 (Value); 

DataTypes => tells JS what type of value we have stored in our variables. There are many data types in JS e.g number, string, boolean, null, undefined, objects, etc.

Operators => used to perform some actions on variables. e.g => + - * / %;
